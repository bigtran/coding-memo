

### sublime 如何配置中文

* 1、 使用 Shift + Ctrl + P，弹出查找栏
* 2、 在搜索框中输入关键字 install，选择 Package Control: Install Package
* 3、 等待一会儿，弹出“信息框”后，输入 chinese，在搜索结果中选择 ChineseLocalizations
* 4、 点击后安装完成，Help -> Language -> 简体中文
